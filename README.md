# Галерея фотографий на PHP

## Описание проекта

Веб-приложение для загрузки и просмотра фотографий с системой подсчета просмотров. Пользователи могут загружать изображения, просматривать галерею и видеть детальную информацию о каждой фотографии.

## Структура файлов

```
/photo_gallery/
├── config.php         - настройки подключения к БД
├── index.php          - главная страница с галереей
├── photo.php          - просмотр отдельной фотографии
├── upload.php         - форма загрузки фотографий
├── style.css          - единый файл стилей
└── uploads/           - папка для загруженных изображений
    └── thumbnails/    - миниатюры фотографий
```

## Структура базы данных

### Таблица `users`
- `id` - уникальный идентификатор
- `name` - имя пользователя
- `email` - электронная почта

### Таблица `photos`
- `id` - уникальный идентификатор
- `user_id` - ID пользователя
- `filename` - имя файла
- `thumbnail` - имя миниатюры
- `title` - название фотографии
- `views` - количество просмотров
- `uploaded_at` - дата и время загрузки

## Основные функции

**index.php:**
- Отображение всех фотографий в виде сетки
- Сортировка по популярности (просмотрам)
- Счетчик общего количества фотографий
- Адаптивный дизайн галереи

**upload.php:**
- Форма загрузки с проверкой данных
- Поддержка форматов: JPG, PNG, GIF
- Автоматическое создание миниатюр (200×200px)
- Регистрация или поиск пользователя по email
- Валидация email и обязательных полей

**photo.php:**
- Просмотр полноразмерного изображения
- Автоматический подсчет просмотров
- Информация об авторе и дате загрузки
- Красивое отображение счетчика просмотров

**style.css:**
- Единый файл стилей для всех страниц
- Градиентные кнопки и карточки
- Эффекты наведения и анимации
- Адаптивная верстка

## Особенности

- Автоматическое создание миниатюр с сохранением пропорций
- Валидация загружаемых файлов
- Красивый современный дизайн с градиентами
- Сохранение прозрачности для PNG изображений

## SQL для создания таблиц

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE NOT NULL
);

CREATE TABLE photos (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    filename VARCHAR(255) NOT NULL,
    thumbnail VARCHAR(255) NOT NULL,
    title VARCHAR(255) NOT NULL,
    views INT DEFAULT 0,
    uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```
